RPG

Arcade 與 Pycairo 製作的桌面 RPG。GitLab Pages 會發布這個靜態下載頁；遊戲本體需要在本機用 Python 執行。

下載 rpg.py requirements.txt Windows 安裝 macOS / Linux 安裝

Windows

powershell -NoProfile -ExecutionPolicy Bypass -File .\setup_game.ps1 .\.venv\Scripts\python.exe rpg.py

macOS / Linux

chmod +x setup_game.sh ./setup_game.sh ./.venv/bin/python rpg.py
